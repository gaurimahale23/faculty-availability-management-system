# Faculty Availability & Leave Tracker

A real-world faculty availability management system built using **Python (CGI)**, **MySQL**, **HTML**, **CSS**, **Bootstrap**, and **Apache (XAMPP)**.  
The system helps administrators manage faculty, allows faculty to update their availability, and enables students to view faculty status.

---

## Features

- Admin login and dashboard
- Add new faculty (Admin only)
- Faculty login and self-update availability (Available / On Leave)
- Admin override faculty status
- Student search and view faculty availability (read-only)
- Integrated and professional UI using Bootstrap
- Live demo possible using Ngrok

---

## User Roles

### Admin
- Login securely
- Add faculty and create faculty login
- View all faculty availability
- Override faculty status when required

### Faculty
- Login using credentials created by admin
- Update only their own availability status
- View status update confirmation on the same dashboard

### Student
- Search faculty by name
- View current availability status
- Read-only access (no login required)

---

## Tech Stack

- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** Python (CGI)  
- **Database:** MySQL  
- **Server:** Apache (XAMPP)
